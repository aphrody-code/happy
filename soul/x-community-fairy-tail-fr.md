## 🔗 Integration avec le Bot Happy

### Strategie de veille
Le script `scripts/fetch-community-tweets.ts` utilise l'API GraphQL de X pour scraper les tweets de la communaute. Il necessite les variables d'environnement :
- `TWITTER_AUTH_TOKEN` — Token d'authentification
- `TWITTER_CT0` — Token CSRF
- `TWITTER_TWID` — ID utilisateur Twitter

### Cas d'usage pour le bot
1. **Veille communautaire** — Surveiller les tendances et sujets chauds de la fanbase FR pour adapter le contenu du serveur Discord
2. **Diffusion cross-plateforme** — Partager les tweets populaires ou actualites dans le salon `📢・annonces` du serveur
3. **Pensees de Happy** — Publier des citations et reactions en personnage sur la communaute X
4. **Engagement** — Repondre aux discussions et relayer les evenements du serveur (quetes RPG, evenements inter-guildes)
5. **Recrutement** — Inviter les membres de la communaute X vers le serveur Discord Fairy Tail FR

### Synergies avec les fonctionnalites du bot
| Fonctionnalite Bot | Synergie X |
|--------------------|------------|
| `/guilde` | Partager les stats de guildes du serveur sur X |
| `/rpg` | Annoncer les nouvelles quetes et lieux RPG |
| `/announce` | Cross-poster les annonces serveur vers X |
| `/happy` | Publier des citations de Happy en reponse aux trending topics |
| Systeme de moderation | Aligner les regles serveur avec celles de la communaute X |
